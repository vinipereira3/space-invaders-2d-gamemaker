## space-invaders-2d-gamemaker
Jogo 2D estilo Space Invaders focado em sistemas de áudio, eventos e estruturação de gameplay utilizando GameMaker.

Projeto inspirado no clássico Space Invaders, desenvolvido no GameMaker.

## Funcionalidades:
- Sistema de tiro
- Inimigos com movimentação
- Efeitos sonoros
- Power-ups

## Destaque
Foco na implementação de áudio para melhorar a experiência do jogador.
